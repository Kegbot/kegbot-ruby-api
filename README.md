# KegbotApi

[Kegbot](https://kegbot.org/) connected kegerator [REST API](https://kegbot.org/docs/api/) client implemented in Ruby. 

## Installation

Add this line to your application's Gemfile:

    gem 'kegbot_api'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install kegbot_api
